# Web Gallery

Full-stack web application that allows users to upload and display images in a centralized online gallery.

## Features

- Image upload functionality
- Dynamic gallery display
- Cloud database integration
- Responsive user interface
- Real-time content updates

## Technologies

- JavaScript
- HTML5
- CSS3
- MongoDB

## Purpose

This project was developed as part of a Web Programming course to demonstrate full-stack development skills, database integration and user interface design.

## Future Improvements

- User authentication
- Image categories and tags
- Search and filtering
- Image compression
- Admin dashboard
